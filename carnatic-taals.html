Suladi Sapta Tala system

The Suladi Sapta Tala system (which was popularized during the time of Purandara Dasa (1484 - 1564) and has been attributed to him) is based on 7 basic taals which employ three basic angas. The ancient system of taals used more angas as listed below, but the Suladi system uses only the first three in the list.
